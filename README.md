# App Mobile para Gestão de Reparos em Patrimônio

Este aplicativo móvel foi desenvolvido utilizando React Native, Expo Bare Workflow, Native Base e Firebase para oferecer uma solução eficaz para o cadastro e gerenciamento de solicitações de reparo e solução em patrimônios. Com uma interface intuitiva e funcionalidades robustas, o aplicativo simplifica o processo de comunicação entre os usuários e a equipe responsável pelos reparos.

## Funcionalidades Principais

- **Cadastro de Solicitações:** Os usuários podem facilmente criar novas solicitações de reparo, incluindo detalhes do problema e localização do patrimônio.

- **Acompanhamento em Tempo Real:** Os usuários podem acompanhar o status de suas solicitações em tempo real, desde o momento do cadastro até a conclusão do reparo.

- **Histórico de Reparos:** O aplicativo mantém um histórico completo de todas as solicitações de reparo, permitindo uma visão geral das atividades anteriores.

- **Notificações:** Os usuários recebem notificações instantâneas sobre o progresso de suas solicitações, garantindo uma comunicação eficiente.

## Tecnologias Utilizadas

- **React Native:** Utilizamos o framework React Native para o desenvolvimento da interface do usuário, garantindo uma experiência consistente em dispositivos Android e iOS.

- **Expo Bare Workflow:** Optamos pelo Expo Bare Workflow para aproveitar ao máximo as funcionalidades nativas dos dispositivos, proporcionando uma performance excepcional.

- **Native Base:** Implementamos o Native Base para criar componentes de interface do usuário elegantes e altamente personalizáveis, garantindo uma aparência moderna e intuitiva.

- **Firebase:** Utilizamos o Firebase para armazenamento de dados em tempo real, autenticação segura de usuários e notificações push em tempo real.

## Como Iniciar

1. **Clone o Repositório:**
`git clone https://github.com/seu-usuario/nome-do-repositorio.git`


2. **Instale as Dependências:**
`cd nome-do-repositorio`
`npm install`


3. **Configuração do Firebase:**
- Crie um projeto no Firebase (https://console.firebase.google.com/).
- Configure as credenciais do Firebase no arquivo `firebaseConfig.js`.

4. **Inicie o Aplicativo:**
`expo start`


5. **Escaneie o Código QR:** Abra o aplicativo Expo Go em seu dispositivo móvel e escaneie o código QR gerado para visualizar o aplicativo em seu dispositivo.
